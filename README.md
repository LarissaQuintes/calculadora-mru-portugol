# Calculadora de MRU em Portugol 🧮

## Sobre o projeto 📌

Este projeto foi desenvolvido com o objetivo de praticar lógica de programação utilizando Portugol. A aplicação funciona como uma calculadora de Movimento Retilíneo Uniforme (MRU), permitindo calcular velocidade, distância ou tempo a partir dos valores informados pelo usuário

## Funcionalidades 💎

- Calcular Velocidade
- Calcular Distância
- Calcular Tempo

## Fórmulas utilizadas 🧭

- V = D / T
- D = V * T
- T = D / V

## Objetivo 👩‍💻

Projeto desenvolvido para prática de lógica de programação e compreensão das fórmulas do MRU.
